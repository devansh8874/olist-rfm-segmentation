# 🛒 Customer Segmentation · Olist RFM Dashboard

A production-ready Streamlit app that performs **RFM (Recency, Frequency, Monetary) analysis** on the [Olist Brazilian E-Commerce dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) from Kaggle to segment customers into actionable groups.

## 📦 Features

| Feature | Details |
|---|---|
| **RFM Scoring** | Quantile-based R/F/M scoring (2–5 configurable bins) |
| **8 Segments** | Champions, Loyal, Promising, Potential Loyalists, At Risk, Needs Attention, Hibernating, Lost |
| **Interactive Charts** | Donut, bar, scatter bubble, histogram — all Plotly |
| **Recommendations** | Per-segment marketing strategy + action items |
| **CSV Upload** | Drop in real Olist CSVs for live analysis |
| **Export** | Download filtered segments as CSV |

## 🧠 RFM Methodology

| Metric | Definition |
|---|---|
| **Recency** | Days since the customer's last purchase (lower = better) |
| **Frequency** | Number of unique orders placed |
| **Monetary** | Total revenue generated by the customer |

Each metric is split into quantiles (default: 4) and scored 1–4. Customers are then classified into 8 segments based on their combined R/F/M scores.

---

## 📊 Segment Logic

```
Champions        → R≥3, F≥3, M≥3
Loyal Customers  → R≥3, F≥2
Promising        → R≥3, F=1
Potential Loyalists → R=2, F≥2
Needs Attention  → R=2, F=1
At Risk          → R=1, F≥3
Hibernating      → R=1, F=2
Lost             → R=1, F=1
```
